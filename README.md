# Audio-Signal-Encryption-and-Decryption-for-Secure-Data-Transfer
This project presents a MATLAB application for safeguarding audio files using digital signal processing techniques. The program supports two independent encryption methods to scramble audio data and prevent unauthorized access.
# Software Used
Open Source: MATLAB
VIT Academic License: MATLAB R2023b (App Designer, DSP Toolbox).
Dataset Name/Source: Sample .wav audio files recorded and tested locally
within the MATLAB environment.
